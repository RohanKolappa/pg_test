Burn to DVD


You can now run the Roxio DVD Burning software to copy the media to a DVD disc.

STEPS TO BURN DVD:

1.) Insert a blank DVD into the DVD recorder drive
2.) Start the Roxio software - Click on the Roxio icon on the desktop
3.) Click on File -> New Project -> Data -> Create Data Disc
4.) Click on Add Data -> Add Folder option and select the folders blob and Player from the source directory for the project
5.) Click on Add Data -> Add Files option and select the files autorun, index.html, IPVSAutoRun, launchie and Medialist.xml from the source directory for the project
6.) Click the "Click button to continue" button in the lower right corner to begin the burning process
7.) Check status....
8.) Save the project if you need to burn DVDs with same folders later.
